Getting Started

The Chaotic TCG API is free to use and utilizes simple HTTP requests to retrieve data from the server. To try it out, open a terminal and make a curl request.

curl http://chaoticga.me/creatures/DOP/15

This should return the data JSON data for the creature card Maxxor in the DOP (Dawn of Perim) set.

{{maxxor | json}} Base URL

Make all API requests to this URL.

http://chaoticga.me/ Authentication

Choatic TCG API is a free API, so there is no authentication required to make use of it. However, since it is free every IP is limited to how many requests can be made over a ceratin amount of time.
